Hostinfo Client
===============================

version number: 0.0.1
author: Dougal Scott

Overview
--------

Command line client to hostinfo

Installation / Usage
--------------------

To install use pip:

    $ pip install hostinfo_client


Or clone the repo:

    $ git clone https://github.com/dwagon/hostinfo_client.git
    $ python setup.py install
